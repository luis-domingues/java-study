Tipos primitivos são: byte, short, int, long, boolean, char, double e float
Onde
   - BYTE: representa inteiros pequenos (-127 à 128). Utiliza 8 bits.
   - SHORT: representa uma quantidade maior de números interios (-32767 a 32767). Utiliza 16 bits.
   - INT: representa uma quantidade ainda maior de números interios (-2147483648 a 2147483647). Utiliza 32 bits.
   - LONG: representa números extremamente grandes (-9223372036854775808 a 9223372036854775807). Utiliza 64 bits.
   - BOOLEAN: representa um valor lógico (true ou false).
   - CHAR: representa um simples caractere entre aspas simples.

O tipo "string" não considerado um tipo primitivo em Java, e sim uma classe, pois  corresponde aos caracteres da tabela ASCII
   - Ela deve ser instaciada com S maiúsculo e seu conteúdo deve ser escrito entre aspas duplas.
   - ex: String meuNome = "Olá, eu sou Luís ";

   - DOUBLE e FLOAT: ambos são usados para resprentar um número com um ponto flutuante. Sendo que, DOUBLE é representa "números flutuantes" maiores, com mais precisão. Já o FLOAT possui uma fase de representação um pouco menor, representando números flutuantes simples e no final é necessario colocar um "f" para diferenciar os tipos.
   - ex: double numeroUm = 3.499899899
         float numeroDois = 3.5F;
